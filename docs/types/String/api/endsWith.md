# [←](../String.md) `String`.`endsWith`

> Returns `true` **if ends with `target`**.

```zig
pub fn endsWith(self: Self, target: []const u8) bool
```

- #### 🧩 Parameters

    | Parameter | Type         | Description              |
    | --------- | ------------ | ------------------------ |
    | `self`    | `Self`       | The `String` instance.   |
    | `target`  | `[]const u8` | The value to search for. |

- #### ✨ Returns : `bool`

    > Returns `true` **if the `String` instance ends with `target`**.

- #### 🧪 Examples

    ```zig
    const String = @import("io").types.String;
    const string = try String.init(allocator, "Hello 👨‍🏭!");
    defer string.deinit();
    ```

    ```zig
    _ = string.endsWith("!");   // 👉 true
    _ = string.endsWith("👨‍🏭");  // 👉 false
    ```
