    - `_self` : `Self`

- #### Returns : `types.cstr`

    > The source of string (`_self.m_buff.?[0.._slef.m_bytes]` or `""`).

<div align="center">

- #### Example

    ```zig
    const string = @import("io").utils.string;
    ```

    ```zig
    var str = string.init();
    defer str.deinit();

    _ = str.src(); // 👉 ""

    str.append("Hello 🌍!");

    _ = str.src(); // 👉 "Hello 🌍!"
    ```
