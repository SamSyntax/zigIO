# [←](../Buffer.md) `Buffer`.`toUpper`

> Converts all (ASCII) letters to uppercase.

```zig
pub fn toUpper(self: *Self) void
```

<img src="https://raw.githubusercontent.com/maysara-elshewehy/io-bench/refs/heads/main/dist/img/md/line.png" alt="line" style="width:500px;"/>

- #### 🧩 Parameters

    | Parameter | Type   | Description            |
    | --------- | ------ | ---------------------- |
    | `self`    | `Self` | The `Buffer` instance. |

- #### ✨ Returns : `void`

    > Modifies the `Buffer` instance in place.

<div align="center">
<img src="https://raw.githubusercontent.com/maysara-elshewehy/io-bench/refs/heads/main/dist/img/md/line.png" alt="line" style="width:500px;"/>

- #### 🧪 Examples

    ```zig
    const Buffer = @import("io").types.bytes;
    var buffer = try Buffer.init(18, "HeLLo 👨‍🏭!");
    ```

    ```zig
    buffer.toUpper(); // 👉 "HELLO 👨‍🏭!"
    ```
